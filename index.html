Distributions

Probability

  • Suppose \(X\) is a random variable; the probability that \(X=a\) is the proportion of times \(X\) take on the value \(a\) in many, many independent realizations of \(X\)

  • For example, consider the random variable \(X\) constructed from a coin flip with potential outcomes \(\{H,T\}\)

    \[X = \left\{ \begin{array}{ll} 1 & \text{if } H, \\ 0 & \text{if } T; \end{array} \right.\]

    then, \(P(X = 1)\) is the proportion of times that \(X=1\) occurred the in many, many coin-flip repetitions

Distributions

  • The distribution of a random variable specifies the probability of realizing each value of the random variable

  • Distinguish b/w discrete and continuous random variables

The probability mass function

  • The probability mass function (PMF) of a discrete random variable \(X\) is a function \(f_X\) such that

    \[f_X(a) = P(X = a); \quad \sum_a f_X(a) = 1\]

  • The PMF is a list of values taken by the random variable \(X\) together with a list of associated probabilities

The cumulative distribution function

  • The cumulative distribution function (CDF) of a random variable \(X\) is a function \(F_X\) such that

    \[F_X(a) = P(X \leq a)\]

The CDF and PDF of a continuous random variable

  • The definition of the cumulative distribution function extends to continuous random variables

  • The PMF is unfit for continuous random variables; instead, introduce the probability density function (PDF)

    \[f_X(a) = \frac{d}{da} F_X(a)\]

  • The area under the PDF \(f_X\) between two points is equal to the probability that \(X\) takes a value between those points

Summary

Summary

  • The distribution of a random variable specifies the probability for each value taken by the random variable

  • The probabilities with which a discrete random variable takes on different values are given by the PMF

  • The probabilities with which a continuous random variable takes on different values can be computed from the PDF

  • The CDF gives the probability that a random variable takes a value less than or equal to a specific value
